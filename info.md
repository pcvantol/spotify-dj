# DJConnect

DJConnect is a Home Assistant custom integration for DJConnect devices.

The integration handles pairing, Spotify OAuth provisioning, device status, Home Assistant Assist/TTS integration, and HA-managed OTA firmware updates. DJConnect devices use Home Assistant Assist websocket STT for microphone audio, then send recognized text commands to the integration.

DJ responses are delivered to the DJConnect device display/speaker through `/api/device/dj_response` with text plus an optional temporary WAV or MP3 audio URL generated by Home Assistant TTS.

This repository includes local branding assets for HACS/Home Assistant display.
