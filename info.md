# SpotifyDJ

SpotifyDJ turns a LilyGO T-Embed ESP32-S3 into a handheld AI Spotify radio DJ remote for Home Assistant.

Features include push-to-talk voice control, Spotify OAuth provisioning, pairing, device registry support, OpenAI DJ announcements, WAV responses, and HA-managed OTA firmware updates.

This repository includes local branding assets for HACS/Home Assistant display.
