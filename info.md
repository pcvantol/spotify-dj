# SpotifyDJ

SpotifyDJ is a Home Assistant custom integration for SpotifyDJ devices.

The integration handles pairing, Spotify OAuth provisioning, device status, Home Assistant Assist/TTS integration, and HA-managed OTA firmware updates. SpotifyDJ devices use Home Assistant Assist websocket STT for microphone audio, then send recognized text commands to the integration.

DJ responses are delivered to the SpotifyDJ device display/speaker through `/api/device/dj_response` with text plus an optional temporary WAV or MP3 audio URL generated by Home Assistant TTS.

This repository includes local branding assets for HACS/Home Assistant display.
