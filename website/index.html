De ervaring uit de doos

SpotifyDJ is ontworpen als een product, niet als een bouwpakket. De firmware staat al op het device; jij installeert de Home Assistant integration en koppelt het device.

📦

Pre-flashed geleverd

Geen USB flash, geen firmware build. Het device komt klaar voor setup in Home Assistant.

🏠

Home Assistant native

Home Assistant regelt pairing, updates, voice verwerking en muziekbediening.

🎙️

Push-to-talk muziek

Druk op de knop, spreek je verzoek in en SpotifyDJ helpt je muziek starten.

Quick start

Setup is bedoeld als: integration installeren, Spotify autoriseren, device pairen, klaar.

  1. Installeer via HACS

    Voeg de SpotifyDJ custom integration toe in HACS en restart Home Assistant.

  2. Voeg SpotifyDJ toe

    Ga naar Settings → Devices & services → Add integration → SpotifyDJ.

  3. Autoriseer Spotify

    Log in met je Spotify Premium account wanneer de setup daarom vraagt.

  4. Pair het device

    Voer de code in die op het SpotifyDJ device verschijnt.

  5. Druk en praat

    Druk op de middelste knop en zeg bijvoorbeeld: “Speel het nieuwste album van Pearl Jam”.

Veelgestelde vragen

Korte antwoorden op de vragen die je waarschijnlijk als eerste hebt.

Moet ik firmware flashen?

Nee. Ga uit van pre-flashed devices. Firmware updates lopen later via Home Assistant OTA.

Heb ik MQTT nodig?

Nee. Nieuwe SpotifyDJ devices werken zonder MQTT.

Speelt DJ audio via Spotify Connect?

Nee. DJ responses spelen op de speaker/display van het SpotifyDJ device.

Kan ik private playlists gebruiken?

Ja, zolang Spotify in Home Assistant goed geautoriseerd is.

Klaar voor je volgende plaat?

Installeer de integration, pair het pre-flashed device en laat Home Assistant de rest doen.
