De ervaring uit de doos

SpotifyDJ is ontworpen als een product, niet als een hobbyproject met losse scripts. De firmware staat al op het device; Home Assistant doet pairing, OAuth, Assist, TTS, OTA en provisioning.

📦

Pre-flashed geleverd

Geen PlatformIO, geen USB flash, geen firmware build. Het device komt klaar voor setup in Home Assistant.

🏠

Home Assistant native

Pairing, status, OTA updates, voice verwerking en device controls lopen via de SpotifyDJ integration.

🎙️

Push-to-talk muziek

Druk op de knop, spreek je verzoek in, HA Assist verwerkt spraak en SpotifyDJ start muziek via het device.

Hoe het werkt

De ESP hoeft geen Home Assistant websocket credentials te kennen. Hij praat met de lokale SpotifyDJ API met een eigen bearer token.

Quick start

Voor de gebruiker voelt setup als: HACS installeren, Spotify autoriseren, device pairen, klaar. De firmware is al aanwezig.

  1. Installeer via HACS

    Voeg de SpotifyDJ custom integration toe in HACS en restart Home Assistant.

  2. Voeg SpotifyDJ toe

    Ga naar Settings → Devices & services → Add integration → SpotifyDJ.

  3. Autoriseer Spotify

    De setup opent Spotify OAuth via Home Assistant. Log in met je Spotify Premium account.

  4. Pair het device

    Voer de pairing code of 12-character device suffix in die op het SpotifyDJ device verschijnt.

  5. Druk en praat

    Druk op de middelste knop en zeg bijvoorbeeld: “Speel het nieuwste album van Pearl Jam”.

Veelgestelde vragen

De korte versie: SpotifyDJ is voor mensen die Home Assistant al gebruiken en Spotify hands-on, fysiek en lokaler willen bedienen.

Moet ik firmware flashen?

Nee. Ga uit van pre-flashed devices. Firmware updates lopen later via Home Assistant OTA.

Heb ik MQTT nodig?

Nee. Nieuwe firmware gebruikt de lokale SpotifyDJ API met bearer token.

Speelt DJ audio via Spotify Connect?

Nee. DJ responses spelen op de speaker/display van het SpotifyDJ device.

Kan ik private playlists gebruiken?

Ja, Spotify OAuth bevat playlist-read-private zodat het device eigen/private playlists kan vinden.

Klaar voor je volgende plaat?

Installeer de integration, pair het pre-flashed device en laat Home Assistant de rest doen.
